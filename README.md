#  MINI GESTION BANCARA
Crea programa en Python que simule un pequeño sistema de gestión bancaria

Nuestro programa permite realizar acciones como:

    Crear una cuenta
    Consultar saldo
    Depositar dinero
    Retirar dinero
    Hacer transferencias
    Ver historial de movimientos


Antes de entrar al banco:

    Usuario + contraseña
    Mensaje de bienvenida 
    Bloqueo de cuenta tras varios intentos


Cada vez que el cliente haga una operación, se registra:

    Fecha
    Hora
    Tipo de operación
    Cantidad
    Saldo final

Cuando el usuario escribe algo mal:
    Mensajes como: “saldo insuficiente”, “opción no válida”, etc.

El sistema puede manejar varias cuentas:
    Registrar usuarios nuevos
    Iniciar sesión con su cuenta
    Cada uno con su saldo independiente

Un menú claro:
    Crear cuenta
    Iniciar sesión
    Depositar
    Retirar
    Transferir
    Ver historial
    Salir
